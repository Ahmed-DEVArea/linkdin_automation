# LinkedIn Content Engine

AI-powered LinkedIn post automation for personal branding.

**Features:** Idea generation, post writing, carousel creation, Higgsfield image generation, Notion storage.

**LLM Support:** Claude, GPT-4o, Gemini.

## Quick Start

```bash
npm install
cp .env.example .env.local  # Add your API keys
npm run dev
```

See [SETUP.md](SETUP.md) for full documentation.
